READ THIS BEFORE USING THE CALCULATOR:
  • This calculator works with polynomials and exponentials. It also works with ln and log10 but not other logarithms unless you use the change of base formula. It also works with sine and cosine, but you must combine them to get the other four trigonometric functions. You can use parentheses to control the order of operations.
  • Type your function into the first box on the left, which says, "enter your function." Don't type y= or f(x)= before your function. Just type the function itself. For instance, if your function is f(x) = x2, just type x2.
  • Enter the lower bound of your Riemann Sum, the upper bound of your Riemann Sum, and the number of rectangles you want to use in your calculations into the respective boxes.
  • In the bottom box, type "right", "left", or "midpoint" depending on whether you want the calculator to take a right Riemann Sum, a left Riemann Sum, or a midpoint Riemann Sum. Make sure to type the word exactly correctly, without any uppercase letters.
  • You must use x as your variable.
  • Use the carrot (^) to represent exponents. For instance, if you want to enter the function x6, type "x^6".
  • To represent π, type "pi" in lowercase letters.
  • Use the asterisk for multiplication. You cannot place values adjacent to each other to represent multiplication. If your equation is 4x2, you must type "4*x^2".
  • Special functions require you to use Javascript Math methods. For trig functions, all this entails is adding a "Math." before your function. See the documentation below. You must type them exactly as they are listed below, otherwise the calculator won't recognize them.
    • To type sin(u), where u is any function, type "Math.sin(u)". If you wanted to enter the function sin(πx + 4), you would type "Math.sin(pi*x+4)".
    • To type cos(u), where u is any function, type "Math.cos(u)". If you wanted to enter the function cos(πx + 4), you would type "Math.cos(pi*x+4)".
    • To type ln(u), where u is any function, type "Math.log(u)". If you wanted to enter the function ln(πx + 4), you would type "Math.log(pi*x+4)".
    • To type log10(u), where u is any function, type "Math.log10(u)". If you wanted to enter the function log10(πx + 4), you would type "Math.log10(pi*x+4)".
  • Note: because Javascript numbers are IEEE 754 floating point numbers, there are inherent inaccuracies in how it stores numbers. If a number has too many decimal places, Javascript will run out of bit storage for decimals and attempt to round the number. This can cause Javascript to evaluate equivalent values as slightly different values. So, if the calculator returns a number like 3.1086244689504383e-15, it is safe to assume that the actual answer is zero, and there was simply a Javascript binary data error. Such errors are unavoidable without completely reworking Javascript's eval framework.
